# Wanderlust
Wanderlust is a travel package booking platform where travel enthusiasts can explore various packages in India and book them at affordable prices with a rich and seamless user experience. The app offers a wide range of customizable packages and is developed using the MERN stack along with several other technologies such as TypeScript, SOLID principles, and the repository pattern. Key features of this project include Stripe payment integration, a real-time chat and video call system, notifications, JWT authentication, map integration, and AWS S3 bucket support.

## Tech Stack 
 ## 🛠 Tech Stack
- **Frontend:** React, TypeScript, TailwindCSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT
- **Cloud Services:** AWS S3
- **Payment Gateway:** Stripe
- **Principles/Patterns:** SOLID, Repository Pattern
